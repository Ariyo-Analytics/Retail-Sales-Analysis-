# Retail Sales Analysis

## Project Overview
This project analyzes retail sales data to uncover trends in revenue, product performance, and regional sales distribution.

The goal of the analysis is to support business decision-making by identifying:
- Top-performing product categories
- Monthly revenue trends
- Regional sales performance
- Opportunities for growth

## Tools Used
- Excel – Data Cleaning and Preparation
- SQL – Data Exploration and Aggregation
- Power BI – Data Visualization and Dashboard Creation

## Key Business Insights

### 1. Product Performance
Phones generated the highest revenue, indicating strong demand for technology products.

### 2. Seasonal Trends
Sales peaked in September and November, suggesting strong Q4 performance and promotional impact.

### 3. Regional Analysis
The West region produced the highest revenue, showing it as the most valuable market.

### 4. Opportunity
Lower-performing categories like Supplies may require pricing or marketing adjustments.
